# 1.4.1 - Use of Color

## Official source

* https://www.w3.org/WAI/WCAG21/Understanding/non-text-content.html

## Official Success criterion

Color is not used as the only visual means of conveying information, indicating an action, prompting a response, or distinguishing a visual element.

Note
> This success criterion addresses color perception specifically. Other forms of perception are covered in Guideline 1.3 including programmatic access to color and other visual presentation coding.

## Team guidelines
* All links have an <u>underline</u>. Usage of contrast is never used to have a link distinguishable from surrounding text. There is only one exception: The surrounding text is underlined. In this case contrast is used to make links distinguishable. **note** keep an eye on minimal contrast requirements (1.4.3 Contrast (Minimum))
